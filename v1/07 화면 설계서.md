# 🖥️ 화면 설계서 (와이어프레임)

> **참고:** 이 문서는 백엔드 개발자가 API 설계와 데이터 흐름을 이해하기 위한 목적으로 작성된 텍스트 기반 와이어프레임입니다.
> 실제 UI/UX 목업은 Figma를 통해 별도 제작합니다. (바이브코딩 활용)

---

## 화면 목록

| # | 화면명 | 접근 권한 | 주요 API 연동 |
|---|--------|-----------|--------------|
| S01 | 메인 홈 | 전체 | GET /api/v1/events, GET /api/v1/events/popular-keywords |
| S02 | 이벤트 검색 결과 | 전체 | GET /api/v1/events/search, GET /api/v2/events/search |
| S03 | 이벤트 상세 | 전체 | GET /api/v1/events/{eventId} |
| S04 | 좌석 선택 | 로그인 | GET /api/v1/sections/{sectionId}/seats, POST /api/v1/seats/{seatId}/hold |
| S05 | 예매/결제 | 로그인 | POST /api/v1/bookings, POST /api/v1/bookings/{id}/pay |
| S06 | 예매 완료 | 로그인 | GET /api/v1/bookings/me |
| S07 | 마이페이지 | 로그인 | GET /api/v1/users/me, GET /api/v1/bookings/me, GET /api/v1/coupons/me |
| S08 | 쿠폰 이벤트 | 로그인 | GET /api/v1/coupons, POST /api/v1/coupons/{id}/issue |
| S09 | CS 문의 채팅 | 로그인 | POST /api/v1/chat/rooms, WebSocket |
| S10 | 어드민 - 이벤트 관리 | Admin | POST/PUT/DELETE /api/v1/admin/events |
| S11 | 어드민 - CS 문의 관리 | Admin | GET /api/v1/chat/rooms, PATCH status |

```
┌─────────────────────────────────────────────────────────┐
│  🔍 "아이유" 검색 결과 42건                              │

│  ┌────────────────────────────────────────────────────┐  │
│  │ 포스터 | 아이유 콘서트 2025            [예매하기]  │  │

│  ┌────────────────────────────────────────────────────┐  │
│  │ ...                                                │  │
```

**백엔드 연동 포인트:**
- WebSocket STOMP 연결: JWT 인증 후 `/sub/chat/{roomId}` 구독

- 이전 내역: `GET /api/v1/chat/rooms/{roomId}/messages?size=20` (커서 페이징)
